A responsive video testimonial slider plugin with customizable colors and easy shortcode integration.

== Description ==

Simple Video Testimonial Slider CWS allows you to create beautiful video testimonial sliders with customizable navigation and play button colors. Features include:

* Responsive design that works on all devices
* Customizable colors for navigation and play buttons
* Easy-to-use shortcode integration
* Multiple video testimonials support
* Smooth animations with Swiper.js

Use the shortcode `[video_testimonials]` to display your testimonial slider anywhere on your site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/simple-video-testimonial-slider-cws` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Video Slider in the admin menu to configure your testimonials.
4. Use the shortcode `[video_testimonials]` in your posts, pages, or widgets.

== Frequently Asked Questions ==

= How do I add testimonials? =

Go to Video Slider > Settings in your WordPress admin. Add video URLs, names, roles, and descriptions for each testimonial.

= What video formats are supported? =

The plugin supports any video format that HTML5 video supports, including MP4, WebM, and OGV.

= Can I customize the colors? =

Yes, you can customize navigation button colors, play button colors, and hover effects in the settings.
